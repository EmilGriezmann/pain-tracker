# CLAUDE.md — Projektanweisungen Pain Tracker

## Arbeitsweise

- **Niemals Schritte ohne ausdrückliche Anweisung ausführen.** Immer warten bis der Nutzer explizit sagt, was als nächstes getan werden soll.
- **Immer eine To-Do-Liste führen** mit `TaskCreate`/`TaskUpdate`, damit der Nutzer transparent sieht, was gerade getan wird. Jeden Task auf `in_progress` setzen wenn er beginnt, auf `completed` wenn er fertig ist.
